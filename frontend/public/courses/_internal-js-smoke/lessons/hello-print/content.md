# Hello, JavaScript

This lesson exists only to validate the multi-language architecture end-to-end.
It is not exposed to learners — the parent course is marked `internal: true`
and filtered out of the dashboard.

Run `main.js` so it prints:

```
Hello, JavaScript!
```

That single `expected_stdout` rule is enough to exercise the schema, loader,
polyglot runner, and validator on a non-Python lesson.
